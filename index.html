VLM-Nav: Monocular Vision Based Mapless Autonomous UAV-Navigation System Guided by Vision Language Model

Gobinda Chandra Sarker*, Md Mehedi Hasan*, Sejuti Rahman
University of Dhaka
Paper Code

Abstract

In this study, we present a novel approach to UAV navigation by integrating vision-based obstacle avoidance with heading control for effective path planning. The proposed system processes RGB images captured by the UAV and converts them into depthmaps using DepthAnything, a zero-shot depth estimator. These depthmaps, alongside the UAV's relative heading angle, are then passed into a vision language model (VLM), which tracks any nearby obstacle and plan motion to avoid it. The output of VLM is used by a neural network to predict the appropriate action. By combining the output of model VLM with the UAV's relative heading, the system dynamically navigates through complex environments towards its destination, effectively avoiding obstacles. Since the system does not rely on any predefined map elements, such as current or target locations, it is completely map-independent. This increases its generalizability, enabling it to adapt to diverse terrains and environments without requiring prior knowledge or map data.
